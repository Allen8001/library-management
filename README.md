# 📚 Library Management System

A backend project built with Java and Spring Boot as part of my journey to become a Java Backend Developer.

---

## 🚀 Project Overview

This project is designed to simulate a real-world Library Management System while learning enterprise Java development.

The goal is to gradually build a production-style backend application using Spring Boot and modern software engineering practices.

---

## 🛠 Tech Stack

- Java 26 *(Planned migration to Java 21 LTS)*
- Spring Boot 4
- Maven
- Git & GitHub
- REST API

Future:
